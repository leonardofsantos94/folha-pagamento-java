# Sistema de Folha de Pagamento em Java

Projeto acadêmico desenvolvido durante o curso de Análise e Desenvolvimento de Sistemas.

## Objetivo

Simular um sistema de folha de pagamento utilizando os conceitos fundamentais da linguagem Java.

## Tecnologias Utilizadas

- Java
- Programação Orientada a Objetos (POO)
- ArrayList
- Scanner

## Funcionalidades

- Cadastro de funcionários
- Cálculo de salários
- Gerenciamento de informações
- Exibição de dados
